🔑 הסבר: הפקת מפתח Google API

המפתח שאתם מפיקים חיוני להפעלת שירות Google Text-to-Speech (TTS). שירות זה הוא הליבה של המערכת, והוא מאפשר המרת טקסט המיילים שלכם לקבצי קול באיכות גבוהה וטבעית.

🛠️ שלבים עיקריים להפקת המפתח

1. כניסה ל-Google Cloud Console

התחילו בכניסה למערכת הניהול של שירותי גוגל. זהו המקום שבו מנהלים את המשאבים הדיגיטליים שלכם בגוגל.

2. בחירת/יצירת פרויקט

כל פעילות ב-Google Cloud מתבצעת תחת פרויקט מסוים. צרו פרויקט חדש עבור שירות זה או בחרו פרויקט קיים שאותו תרצו לשייך לשירות.

3. הפעלת שירותי ה-API הנדרשים

עליכם להפעיל באופן מפורש את שני השירותים הבאים בתוך APIs & Services:

  • Cloud Text-to-Speech API: השירות שממיר את הטקסט לקול.
  • Cloud Translation API: נדרש לטיפול במיילים שאינם כתובים בעברית.
4. יצירת מפתח API

נווטו אל APIs & Services > Credentials (ממשקים ואישורים). שם, צרו מפתח API (API Key).

5. העתקה ושימוש

העתיקו את המפתח שנוצר והדביקו אותו במקום המיועד לכך ביישום.
